💻 Portfolio

A modern, responsive personal portfolio website showcasing my work, skills, and experience as a Full Stack Developer. Built with a focus on clean UI, smooth animations, and performance.

🌐 Live Preview

You can host this project using GitHub Pages or any static hosting service.

✨ Features
🎨 Modern dark-themed UI with purple accent design

⚡ Smooth scrolling and scroll-reveal animations

⌨️ Typewriter text effect for dynamic role display

🌌 Animated particle background using HTML Canvas

📱 Fully responsive (mobile-friendly with hamburger menu)

🧠 Skills section with interactive tabs

🗂️ Project showcase with GitHub & live links

📬 Contact form with simple client-side handling

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (Vanilla)

Styling: Custom CSS (no frameworks)

Fonts: Google Fonts (Raleway & Source Code Pro)

Animations: CSS + JavaScript (IntersectionObserver)

Graphics: HTML Canvas (particle system)

📁 Project Structure

portfolio/
│── index.html   # Main website file (HTML, CSS, JS all in one)

🚀 Getting Started

Clone the repository:
https://github.com/RipMickel/Portfolio.git

Open the project:
cd your-portfolio

Run locally:
Simply open index.html in your browser

📌 Sections Overview

Home – Introduction with typewriter animation

About – Personal background and quick facts

Skills – Categorized tech stack (Frontend, Backend, Tools)
